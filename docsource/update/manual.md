# Manual Update

!!! info

    Blazam can update itself, but some configuration
    is required. Check the [self-update](self.md) page for more information.

Manual updating is very easy with Blazam:

1. [Download the latest release](https://blazam.org/download), alternatively download the latest web installer if you used it to install.
1. Stop your Blazam instance via IIS or Services.
1. Extract the contents of the release zip file to the application directory. If you installed with the web installer, you can run it again to apply the update.
1. Restart Blazam.

All database updates are applied automatically when the application starts
